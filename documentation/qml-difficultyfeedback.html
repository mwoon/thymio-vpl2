DifficultyFeedback QML Type

DifficultyFeedback represents an interface to ask a user for their subjective feeling about an exercise. More...

Import Statement: import
  • List of all members, including inherited members

Detailed Description

This interface can be displayed after every exercise by toggling a variable in GameInterface. It poses the question "How difficult was the task for you?" to the user. The 5 point scale goes from 1 to 5 but internally uses the list index which goes from 0 to 4.
