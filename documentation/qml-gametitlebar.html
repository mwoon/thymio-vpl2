Detailed Description

GameTitleBar defines the symbol and logic of the menu in GameInterface. It displays a hamburger icon in a circle. When clicked on, it opens a drawer on the left with the menu options.

Signal Documentation

openDrawer()

This signal notifies listeners that the icon has been clicked on.
