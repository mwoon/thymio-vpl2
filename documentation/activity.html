    Detailed Description

    The Activity class encapsulates an item in the exercise graph of the ZPDES algorithm.

    The following are default values:

    • id: none
    • bandit level / learning rate: 0.5
    • activation threshold: 0
    • description: none

    As of the implementation when this documentation was written, only the id and bandit level are relevant for the zpdes.

    Member Function Documentation

    Activity::Activity()

    Creates a default Activity with no id and learning rate 0.5

    Activity::Activity(std::string id)

    Creates a default Activity with the given id and learning rate 0.5

    Activity::Activity(std::string id, double at)

    Creates a default Activity with the given id and learning rate 0.5. the parameter at sets the activation threshold.

    Activity::Activity(std::string id, double at, std::string desc)

    Creates a default Activity with the given id and learning rate 0.5. the parameter at sets the activation threshold. desc sets a description for the object. This is the constructor used in the Zpdes.

    Activity::Activity(std::string id, double at, std::string desc, std::list<std::string> exs)

    Creates a default Activity with the given id and learning rate 0.5. the parameter at sets the activation threshold. desc sets a description for the object. exs specifies a list of exercises in this activity, however this can be omitted as it can be included in the id of the activity.

    double Activity::getAverageSuccess()

    Returns the average success of this activity based on the number of times it has been played and the scores.

    void Activity::setDescription(std::string desc)

    Sets a new description for the activity.

    void Activity::updateBanditLevel(double result)

    Updates the bandit level / learning rate of the activity based on the last d*2 activities (d = 5 as of this writing).
